# Lead-Loom

## 一个处理拼豆图片转换并统计豆子数量的简单工具

- 可以将图片转换为像素图案
- 统计多张图片的的像素总数
- 支持手动修改像素点
- 统计多张图片的像素点数
- 导出转换后的像素图片以及统计报告


## A lightweight fuse bead pattern generator that converts images into pixel art designs and automatically counts the required beads.

### Key Features:

- Image-to-pixel-art conversion
- Bead quantity statistics for single or multiple images
- Manual pixel editing and pattern adjustment
- Pattern export support
- Fast material estimation for bead art projects

Designed for fuse bead crafting, pixel art design, and hobbyist makers.
